# Roma Segreta

A family afternoon loop through hidden Rome — one self-contained web app, no build step.

**Live app:** https://vulkanr.github.io/roma-segreta/

Five stops (14:30 → ~17:10, plus an optional fork): Caesar's assassination site run by
cats, a hidden frescoed courtyard, the underground City of Water beneath the Trevi
Fountain, a gelato dare, and either the Capuchin bone crypt or a shopping run
(Via Margutta marble workshop / Monti vintage). Includes a teen photo hunt, per-stop
TikTok concepts, a niche food plan, and practical tips — hours and prices verified
July 2026.

Progress is saved in each phone's `localStorage`. Light/dark theme follows the device.

Developed in the `ai-dream-team` monorepo (`projects/rome-day-trip`); this repo exists
to serve the app via GitHub Pages — deployment runs automatically on every push to
`main` via `.github/workflows/pages.yml`.
